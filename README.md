# django-webpack

- python3 的 django 模板 与 前端 webpack(4/5)打包的解决方案

## 环境

- 后端

  - 安装依赖

    - pip3 install -r requirements.txt

  - 启动服务

    - cd django-webpack/portal/ && pathon3 manage.py runserver

- 前端

  - 安装依赖

    - cd django-webpack/portal/frontend && yarn
  
  - 启动环境（目前只有watch）

    - yarn watch
